# 📄 PDF to Word Converter – Web Application

PDF to Word Converter is a **modern and easy-to-use web application** that allows users to convert PDF documents into editable Word files (.docx). The app provides a clean, responsive interface with smooth interactions, making document conversion fast, accurate, and hassle-free.

The application is ideal for students, professionals, and businesses who need to extract content from PDFs and edit it in Word. It focuses on usability, speed, and a minimal, professional design.

---

## 🚀 Features

• Upload PDF files for conversion  
• Convert PDF documents to editable Word (.docx) format  
• Download converted Word files easily  
• Responsive design for desktop, tablet, and mobile  
• Drag-and-drop file upload support  
• Fast and accurate conversion  
• Clean and minimal user interface  

---

## 🛠 Technologies Used

• HTML5, CSS3, JavaScript  
• React JS (Optional for SPA version)  
• File API / Blob handling for client-side conversion  
• Backend (Optional): Node.js / Python Flask for server-side conversion  
• Libraries: **pdf-lib**, **mammoth.js** (for PDF to Word conversion)  

---

## 📱 Screens / Pages

• Home Page with file upload  
• Conversion progress indicator  
• Download page for converted Word file  
• Responsive UI for multiple devices
